# 🚨 MedAlert – Intelligent Emergency Medical Alert System

MedAlert is an Android-based emergency response application designed to provide instant and intelligent assistance during medical emergencies. It enhances traditional SOS systems by integrating smart detection, real-time tracking, and structured medical reporting.

## 🛡️ Overview
MedAlert transforms a smartphone into a proactive safety system. Instead of sending a basic SOS message, it sends a structured medical report along with real-time location and emergency severity, enabling faster and more effective response.

## 🚀 Features
- Manual SOS with countdown safety
- Voice-based SOS activation
- Shake detection for emergency triggering
- Smart Condition Detection (Normal / High Risk / Critical)
- Live GPS location tracking
- Low Battery Emergency Mode
- Medical profile storage (blood group, allergies, conditions)
- Alert history tracking
- Dark and Light mode UI

## 🧠 Unique Features
- Multi-channel SOS triggering (manual, voice, shake)
- Intelligent emergency classification
- Structured medical report instead of basic alert
- Works under low battery and limited connectivity
- Sensor-based real-time emergency detection

## 🏗️ Tech Stack
- Language: Java
- UI: XML
- Database: SQLite
- Platform: Android Studio
- APIs:
  - SMS Manager API
  - Location Manager (GPS)
  - Speech Recognition API
